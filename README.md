# Flight Mechanics & Aircraft Performance

This repository contains engineering analyses and computational implementations developed as part of my flight training at the Flight Laboratory, Department of Aerospace Engineering, IIT Kanpur.

The objective of this repository is to document the engineering methodology, reproduce the calculations programmatically, and present the results in a reusable and transparent manner.

The focus of this repository is on aircraft performance analysis and flight mechanics, using experimental flight data and standard aerospace engineering methods. Each study includes the theoretical background, data processing, engineering calculations, and Python implementations used to reproduce the results.

## Topics Covered
- Aircraft Weight & Center of Gravity (CG) Analysis
- Mean Aerodynamic Chord (MAC) Calculation
- Longitudinal CG Variation with Passenger Loading
- Drag Polar Estimation
- Cruise Performance Analysis
- Rate of Climb and Angle of Climb Estimation
- Aircraft Performance Visualization using Python
